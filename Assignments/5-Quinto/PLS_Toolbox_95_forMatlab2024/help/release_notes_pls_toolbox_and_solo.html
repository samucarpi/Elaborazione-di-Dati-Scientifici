Version Inter-compatibility

The following table indicates the version compatibility of Eigenvector Research's various Matlab-based products. For details on inter-compatibility between Solo_Predictor or Model_Exporter and PLS_Toolbox/Solo, see the Solo Predictor Release Notes and the Model Exporter Release Notes.

PLS_Toolbox and Add-on Compatibility Table

MATLAB-based products inter-compatibility. PLS_Toolbox is compatible with MATLAB versions newer than 5 years prior to release date. The MIA_Toolbox or EMSC_Toolbox versions indicated in the third and fourth columns are the minimum version required for use with the corresponding PLS_Toolbox. Versions of MIA_Toolbox and EMSC_Toolbox older than the version specified will not work with the PLS_Toolbox version indicated. *E = as of version 7.5, EMSC_Toolbox is included with PLS_Toolbox and the separate EMSC_Toolbox is not needed.

NOTE: For Solo based products the operating system compatibility will track the version of MATLAB used to build Solo (column 6 below). Use the MATLAB platform road-map to see compatibility. For system requirements see specific versions of MATLAB see here. The MATLAB Runtime is automatically installed by the Solo installer.
