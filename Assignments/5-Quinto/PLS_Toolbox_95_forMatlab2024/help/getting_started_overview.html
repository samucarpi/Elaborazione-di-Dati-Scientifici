Thanks for your interest in Eigenvector Research software. We know that getting familiar with new software can be a time-consuming task, so in addition to providing various forms of documentation, we've tried to collect some of the most critical links to getting started information on this page. As always, if you have questions, feel free to contact our helpdesk with questions.

Using the Software

Need help getting started using the software? Here are some sources of information, by product. Plus, the following are relevant to all products:

  • EigenGuide How-to videos: These free videos show the basics of working with our software including how to make PCA and PLS models, plus videos on the basic interface and more advanced methods.
  • Searchable Wiki Documentation: Searchable documentation for all our products is available on our Documentation Wiki.


Solo and PLS_Toolbox

PLS_Toolbox and Solo include many graphical user interfaces (GUIs) which guide the user through various analysis tasks. The following links are good places to start learning how to use those interfaces:

  • Software User Guide: The main page describing the software including installation instructions and several Quick Start guides.
  • Solo/PLS_Toolbox Environment: How-to for the most frequently used interfaces in Solo and PLS_Toolbox.


PLS_Toolbox

In addition to the interfaces mentioned above, PLS_Toolbox also includes a large suite of MATLAB command-line tools. The following links are good places to start learning about how to use the PLS_Toolbox command-line functions:

  • Function Reference Manual: A task-oriented list of functions and their use.
  • Getting Started Documentation and Help: Topics specific to getting to know PLS_Toolbox.
  • Getting Started With Matlab: A basic introduction to using the MATLAB environment.


Image Analysis: Solo+MIA and MIA_Toolbox

In addition to all the links above, the image-enabled products contain additional tools specific to analyzing image data. The following links are good places to start learning to use these tools:

  • Solo+MIA/MIA_Toolbox User Guide: The main page describing the software including installation instructions and several Quick Start guides.
  • The MIA_Toolbox Environment: How-to for the most frequently used image-specific interfaces.

Learning Chemometrics/Multivariate Analysis

At Eigenvector Research, we believe that an educated user is a good user. Although many tasks can be completed without detailed understanding of the numerical methods "under the hood", we think users will make better decisions and learn more. The following are some resources we provide to help you learn about the multivariate methods in the software.

  • EigenGuide How-to videos: These free videos show the basics of working with our software including how to make PCA and PLS models, plus videos on the basic interface and more advanced methods.
  • Chemometrics Tutorial: Our software comes with a comprehensive tutorial on chemometrics. This can be accessed from Solo or PLS_Toolbox via the Help > Chemometrics Tutorial menu in either the Workspace Browser or Analysis interfaces. From PLS_Toolbox, you can also access this through the command:
pls_toolboxhelp
  • EigenU Online: We've made our most popular short courses available on-line. If you really want to understand the statistical methods behind the software, this is a great way to learn from the your own desk.
  • Eigenvector University: Our six-day series of short courses plus other events, including User Poster Session and PowerUser Tips & Tricks Session is a great way to immerse yourself in the field and gain the background you need to be successful with difficult data. Typically attracting 40+ participants, it is also a great place to network with your colleagues about common problems and solutions.
