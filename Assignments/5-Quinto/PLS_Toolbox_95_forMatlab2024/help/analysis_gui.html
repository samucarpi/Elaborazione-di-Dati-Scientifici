Analysis Window

The Analysis window serves as the core interface to the Solo data modeling and analysis functions. You use the Analysis window to create, apply, analyze and explore models.

Using the Analysis window

You can use the following references to locate information on using the Analysis window:

  • Analysis Window Components describes the different components of the user interface.
  • Analysis Menu describes the different methods which are available in the Analysis window.
  • Analysis Phases Overview describes the procedures used to analyze data and create models, including the:
    • Calibration phase
    • Test and Validation phase
    • Model Application phase

Related Tools

Several other data and model manipulation and analysis tools (accessible through the Analysis window menus and the Status pane) are described in additional sections:

  • DataSet Editor Tool - to edit data
  • Preprocessing Tool - to choose data pre-treatments (preprocessing)
  • Correlation Map Tool - to examine variable correlations
  • Cross-Validation Tool - to validate models
  • Permutation Test Tool - to validate models
  • Model Robustness Tool - to validate models

Analysis Window Components

The Analysis window has seven major components. These components include (click on component name for more information):

  • Main menu
  • Toolbar
  • Status pane
  • Control pane
  • Help pane
  • Flowchart pane
  • Model Cache pane
