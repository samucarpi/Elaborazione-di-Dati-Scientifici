Model_Exporter and PLS_Toolbox Compatibility Table

The Model_Exporter version in the first column is the minimum required to use a model from the PLS_Toolbox version listed in the second column. In general, the given version of Model_Exporter must be used with the corresponding version(s) of PLS_Toolbox (backwards compatibility is provided by PLS_Toolbox).

This table does not apply to Solo+Model_Exporter which assures backwards compatibility for any model created or readable by a given version of Solo.
