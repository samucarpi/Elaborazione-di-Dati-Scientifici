Changes and Bug Fixes in Version 7.9.3

File Comment


analysis
  • Updates to Flow Chart for SVMDA and ASCA.
corrmap
  • Fix for saving data with wrong labels when corrmap is using reordering.
exportfigure
  • Add better logic for removing temporary labels and information before exporting.
gselect
  • Multiple fixes for performance and stability.
modeloptimizer
  • Fixes for loading models.
modeloptimizergui
  • Fixes for displaying and comparing custom cross-validation.
netcdfreadr
  • Allow for other than unit mass resolution during the binning process, updates to help.
parsemixed
  • Fix for importing XLS files with more label sets than 1/2 the maximum number of rows or columns.


pereadr
  • ﻿﻿﻿ Transpose imported file's dataset so samples are rows and wavelengths are columns. Throw error if importing multiple files which have wavelength axisscales which are not identical.
